# File transfer

### Deatils
点对点文件传输程序

### Feature
* C99
* SOCKET,winsock2,SOCK_STREAM

### Environment 
* VC++
* IDE : Visual Studio 2015

### Instruct
* 使用方法：
    1. 运行服务器```server.c```(指定文件位置)...
    2. 运行客户端```client.c```(指定接收位置)...
    3. 等待传输过程结束，程序自动退出

* *部分参数可以在源码开头的宏定义处修改*
* **无法穿透NAT**
